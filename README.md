# RCCMS - Revenue Court Computerized Management System

🏛️ **Assistant Collector (Fastrack), Niwai**  
📍 Government of Rajasthan

## 🆕 Updated Features (May 2026)

### 📈 Disposal MPR Report (NEW)
- **From Date - To Date** wise disposal report generation
- Age-wise disposal breakup:
  - 🔴 Over 5 Years
  - 🟠 Over 3 Years  
  - 🟡 Over 2 Years
  - 🟢 Over 1 Year
  - 🔵 Less Than 1 Year
- Detailed file list with pending days at disposal time
- Export to CSV
- Print-friendly format

### 📊 Pending MPR Report
- Existing MPR for pending files
- Age-wise pending breakup

### 📝 File Registration
- Add new RRCMS files
- Auto-save to browser database

### ✅ File Disposal
- Dispose pending files
- Auto-update database

## 📁 Files
| File | Purpose |
|------|---------|
| `index.html` | Main HTML structure |
| `style.css` | Styling & responsive design |
| `data.js` | Pre-loaded data |
| `script.js` | All JavaScript functionality |

## 💾 Data Storage
All data stored in **Browser LocalStorage** - persists across sessions.

## 🚀 How to Use
1. Open `index.html` in any modern browser
2. No server required - runs entirely client-side
3. Data auto-saves to browser storage
